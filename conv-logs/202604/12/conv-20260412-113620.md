## 1. 타이틀바를 정확한 #ffffff + 진한 파랑으로 truecolor 전환
사용: Edit, Bash

Toby: 타이틀바 글씨는 white(#ffffff)로 하고 배경은 좀 더 진한 파란색으로

Claude:
- ANSI 8색 팔레트(`\x1b[37m`, `\x1b[104m`)는 터미널 테마에 따라 다르게 렌더되므로 **24-bit truecolor**로 전환
- `src/ui.ts`:
  - `BG.deepBlue: \x1b[48;2;30;58;138m` 추가 — Tailwind blue-900(#1e3a8a) 기준
  - `FG_PURE_WHITE: \x1b[38;2;255;255;255m` 상수 추가 — 정확한 #ffffff
  - 타이틀바 렌더 코드 업데이트: `BG.brightBlue + FG.white` → `BG.deepBlue + FG_PURE_WHITE + BOLD`
- Runtime 검증 — 생성된 ANSI 시퀀스:
  ```
  \e[48;2;30;58;138m\e[38;2;255;255;255m\e[1m Claude Code Monitor ... \e[0m
  ```
- README: Screenshot 주석을 `[bright blue bg, white bold]` → `[#1e3a8a bg, #ffffff bold]`로 수정, Visual highlights에도 hex 코드 명시

**왜 Tailwind palette**: 수동으로 RGB 튜닝보다 이미 calibrated된 color system을 재사용. blue-900은 "진한 파랑"의 표준적 선택.

**24-bit truecolor의 가치**: iTerm2/VS Code/WezTerm/Alacritty/Kitty 등 모던 터미널 모두 지원. 사용자가 `#ffffff`를 명시한 것은 modern terminal 사용 신호.
